## Introduction

Use this workflow to expand a single page into a wider set of related search queries and angles you can target.

### Here goes the prompt

```md className=wlx-send-to-agent
Provide a query fan out for the web-page: [url]
```

This workflow helps you:

- Generate related query variations from one URL
- Identify adjacent search intent opportunities
- Build a stronger content and optimization plan around the page
